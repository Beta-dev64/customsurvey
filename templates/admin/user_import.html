CSV Format Instructions

Your CSV file should have the following columns:

  • username (required) - Unique username for login
  • password (required) - Initial password
  • full_name (required) - Full name of the user
  • role (required) - Either "admin" or "field_agent"
  • region (required) - Region code (SW, SE, SS, NC, NW, NE, or ALL)
  • state (optional) - State name
  • lga (optional) - Local Government Area
The first row of your CSV should contain the column headers.
Example CSV Format
username password full_name role region state lga
johndoe pass123 John Doe field_agent SW LAGOS Ikeja
janesmith pass456 Jane Smith field_agent NE BORNO Maiduguri
adminuser admin789 Admin User admin ALL
