CSV Format Instructions

Your CSV file should have the following columns:

  • urn (required) - Unique Reference Number
  • outlet_name (required) - Name of the outlet
  • region (required) - Region code (SW, SE, NC, NW, NE)
  • customer_name (optional) - Customer name
  • address (optional) - Address
  • phone (optional) - Phone number
  • outlet_type (optional) - Type (Shop, CONTAINER, Pallet)
  • local_govt (optional) - Local government
  • state (optional) - State
The first row of your CSV should contain the column headers.
